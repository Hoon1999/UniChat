## 프로젝트 개요
프로젝트명: knung
개발 기간: 2024.03.15 ~ 2024.06.16
주제: 웹 채팅 어플리케이션
개요: 주로 사용하는 SNS 채팅 어플리케이션을 모두 프로그램 설치가 필요하여 공공 PC 를 사용할 때 마다 프로그램 설치가 필요하다는 불편함이 있음.
이러한 불편함을 해결하기 위해 프로그램 설치가 필요없는 웹 채팅 프로그램을 개발
